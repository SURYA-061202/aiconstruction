# BuildGuard: AI-Powered Construction Validation

BuildGuard is a professional construction compliance platform built with **React** and powered by the **RCK Engine**. It automates the validation of architectural drawings and ensures safety compliance with zero hallucinations.

## 🚀 Quick Start

1.  **Clone & Install**:
    ```bash
    npx create-vite-app buildguard --template react
    cd buildguard
    npm install axios lucide-react framer-motion
    ```
2.  **Configure RCK**: Follow the [Setup Guide](./docs/setup.md).
3.  **Integrate AEC Checks**: See [Integration Details](./docs/integration.md).

## 🛠 Features

- **Blueprint Extraction**: Automated metadata and dimension capture.
- **UK/Global Compliance**: Real-time validation against Building Regulations.
- **Hallucination-Free Reporting**: Every safety check is backed by NSVG evidence.
- **Compliance Dashboard**: Premium UI for risk management and reporting.
